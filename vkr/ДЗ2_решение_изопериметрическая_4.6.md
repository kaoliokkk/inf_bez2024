# Решение изопериметрической задачи 4.6

**Условие:**
- Функционал: `J = ∫₀¹ x'²dt → extr`
- Изопериметрическое ограничение: `∫₀¹ tx dt = 0`
- Краевые условия: `x(0) = -4`, `x(1) = 4`

## Решение:

### Шаг 1: Метод множителей Лагранжа

Функционал Лагранжа:
```
J_λ = ∫₀¹ (x'² + λtx)dt
```

где `λ` - множитель Лагранжа.

### Шаг 2: Уравнение Эйлера

Для функционала `J_λ = ∫ (x'² + λtx)dt` уравнение Эйлера:
```
d/dt(∂(x'² + λtx)/∂x') - ∂(x'² + λtx)/∂x = 0
```

Вычисляем частные производные:
- `∂(x'² + λtx)/∂x' = 2x'`
- `∂(x'² + λtx)/∂x = λt`

Подставляем:
```
d/dt(2x') - λt = 0
2x'' - λt = 0
x'' = λt/2
```

### Шаг 3: Решение дифференциального уравнения

Общее решение:
```
x'(t) = ∫ (λt/2)dt = λt²/4 + C₁
x(t) = ∫ (λt²/4 + C₁)dt = λt³/12 + C₁t + C₂
```

### Шаг 4: Использование краевых условий и ограничения

**Условие 1:** `x(0) = -4`
```
C₂ = -4
```

**Условие 2:** `x(1) = 4`
```
λ/12 + C₁ - 4 = 4
λ/12 + C₁ = 8
C₁ = 8 - λ/12    (1)
```

**Изопериметрическое ограничение:** `∫₀¹ tx dt = 0`
```
∫₀¹ t(λt³/12 + C₁t - 4)dt = 0
∫₀¹ (λt⁴/12 + C₁t² - 4t)dt = 0
[λt⁵/60 + C₁t³/3 - 2t²]₀¹ = 0
λ/60 + C₁/3 - 2 = 0
λ/60 + C₁/3 = 2    (2)
```

Подставляя (1) в (2):
```
λ/60 + (8 - λ/12)/3 = 2
λ/60 + 8/3 - λ/36 = 2
λ/60 - λ/36 = 2 - 8/3
(3λ - 5λ)/180 = (6 - 8)/3
-2λ/180 = -2/3
-λ/90 = -2/3
λ = 60
```

Из (1):
```
C₁ = 8 - 60/12 = 8 - 5 = 3
```

### Шаг 5: Экстремаль

```
x*(t) = 60t³/12 + 3t - 4 = 5t³ + 3t - 4
```

### Шаг 6: Определение типа экстремума

Вычислим значение функционала:
```
x*'(t) = 15t² + 3
```

```
J[x*] = ∫₀¹ (x*')²dt = ∫₀¹ (15t² + 3)²dt
= ∫₀¹ (225t⁴ + 90t² + 9)dt
= [45t⁵ + 30t³ + 9t]₀¹
= 45 + 30 + 9 = 84
```

Вторая вариация положительна, следовательно, экстремаль доставляет абсолютный минимум.

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = 5t³ + 3t - 4
```

**Значение функционала:**
```
S_max = +∞
```
