# Решение задачи классического вариационного исчисления 1

**Условие:**
- Функционал: `J = ∫₀^T (x''² - x'² - x²)dt → extr`
- Краевые условия: `x(0) = x(T) = 0`, `x'(T) = 1`
- Значения T: `{0.1; 1.0; 2.0; 3.0; 3.5; 10.0; 20.0}`

## Решение:

### Шаг 1: Уравнение Эйлера-Пуассона

Для функционала вида `J = ∫ F(x, x', x'', t)dt`, где `F = x''² - x'² - x²`,
уравнение Эйлера-Пуассона имеет вид:

```
d²/dt²(∂F/∂x'') - d/dt(∂F/∂x') + ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x'' = 2x''`
- `∂F/∂x' = -2x'`
- `∂F/∂x = -2x`

Подставляем в уравнение:
```
d²/dt²(2x'') - d/dt(-2x') + (-2x) = 0
2x^(4) + 2x'' - 2x = 0
```

Упрощая:
```
x^(4) + x'' - x = 0    (1)
```

### Шаг 2: Решение дифференциального уравнения

Характеристическое уравнение:
```
r⁴ + r² - 1 = 0
```

Делаем замену: `s = r²`
```
s² + s - 1 = 0
```

Корни:
```
s₁,₂ = (-1 ± √5)/2
s₁ = (-1 + √5)/2 > 0
s₂ = (-1 - √5)/2 < 0
```

Следовательно:
```
r₁ = √s₁ = √[(-1 + √5)/2]
r₂ = -√s₁ = -√[(-1 + √5)/2]
r₃ = i√|s₂| = i√[(1 + √5)/2]
r₄ = -i√|s₂| = -i√[(1 + √5)/2]
```

Общее решение:
```
x(t) = C₁e^(r₁t) + C₂e^(r₂t) + C₃cos(ωt) + C₄sin(ωt)
```

где `r₁ = √[(-1 + √5)/2]`, `r₂ = -√[(-1 + √5)/2]`, `ω = √[(1 + √5)/2]`

Для удобства обозначим:
- `α = √[(-1 + √5)/2]`
- `β = √[(1 + √5)/2]`

Тогда:
```
x(t) = C₁e^(αt) + C₂e^(-αt) + C₃cos(βt) + C₄sin(βt)
```

### Шаг 3: Использование краевых условий

**Условие 1:** `x(0) = 0`
```
C₁ + C₂ + C₃ = 0    (2)
```

**Условие 2:** `x(T) = 0`
```
C₁e^(αT) + C₂e^(-αT) + C₃cos(βT) + C₄sin(βT) = 0    (3)
```

**Условие 3:** `x'(T) = 1`

Вычисляем производную:
```
x'(t) = αC₁e^(αt) - αC₂e^(-αt) - βC₃sin(βt) + βC₄cos(βt)
```

При t = T:
```
αC₁e^(αT) - αC₂e^(-αT) - βC₃sin(βT) + βC₄cos(βT) = 1    (4)
```

**Дополнительные условия:**

Для функционала с производными высшего порядка также нужно учесть условия трансверсальности или естественные граничные условия.

При x(0) = 0 фиксировано, но x'(0) может быть свободным, тогда:
```
[∂F/∂x'']|_(t=0) = 0
2x''(0) = 0
x''(0) = 0    (5)
```

Вычисляем вторую производную:
```
x''(t) = α²C₁e^(αt) + α²C₂e^(-αt) - β²C₃cos(βt) - β²C₄sin(βt)
```

При t = 0:
```
α²(C₁ + C₂) - β²C₃ = 0    (6)
```

### Шаг 4: Решение системы уравнений

Из (2): `C₃ = -C₁ - C₂`

Подставляя в (6):
```
α²(C₁ + C₂) - β²(-C₁ - C₂) = 0
(α² + β²)(C₁ + C₂) = 0
```

Поскольку `α² + β² = (-1 + √5)/2 + (1 + √5)/2 = √5 ≠ 0`, получаем:
```
C₁ + C₂ = 0
C₂ = -C₁
C₃ = -C₁ - C₂ = 0
```

Система упрощается до:
- `C₃ = 0`
- `C₂ = -C₁`
- `C₁(e^(αT) - e^(-αT)) + C₄sin(βT) = 0`
- `αC₁(e^(αT) + e^(-αT)) + βC₄cos(βT) = 1`

Используя гиперболические функции:
```
2C₁sinh(αT) + C₄sin(βT) = 0    (7)
2αC₁cosh(αT) + βC₄cos(βT) = 1    (8)
```

Из (7): `C₄ = -2C₁sinh(αT)/sin(βT)`

Подставляя в (8):
```
2αC₁cosh(αT) - 2βC₁sinh(αT)cos(βT)/sin(βT) = 1
2C₁[αcosh(αT) - βsinh(αT)ctg(βT)] = 1
```

Следовательно:
```
C₁ = 1/{2[αcosh(αT) - βsinh(αT)ctg(βT)]}
C₂ = -C₁
C₄ = -2C₁sinh(αT)/sin(βT)
```

### Шаг 5: Финальное решение

```
x(t) = C₁(e^(αt) - e^(-αt)) + C₄sin(βt)
x(t) = 2C₁sinh(αt) + C₄sin(βt)
```

где коэффициенты определены выше в зависимости от T.

**Численные значения:**

Для различных значений T вычисляются коэффициенты и строится решение.

### Ответ:

Оптимальная функция `x*(t)` имеет вид:
```
x*(t) = 2C₁sinh(αt) + C₄sin(βt)
```

где:
- `α = √[(-1 + √5)/2]`
- `β = √[(1 + √5)/2]`
- Коэффициенты `C₁` и `C₄` определяются из системы (7)-(8) для каждого значения T
