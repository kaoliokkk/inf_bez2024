# Решение задачи классического вариационного исчисления 2

**Условие:**
- Функционал: `J = ∫₀^T (x''² - x²)dt → extr`
- Краевые условия: `x(0) = x(T) = 0`, `x'(T) = 1`
- T ∈ [0.1; 10.0]

## Решение:

### Шаг 1: Уравнение Эйлера-Пуассона

Для функционала `J = ∫ F(x, x', x'', t)dt`, где `F = x''² - x²`,
уравнение Эйлера-Пуассона:

```
d²/dt²(∂F/∂x'') - d/dt(∂F/∂x') + ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x'' = 2x''`
- `∂F/∂x' = 0`
- `∂F/∂x = -2x`

Подставляем:
```
d²/dt²(2x'') - 0 + (-2x) = 0
2x^(4) - 2x = 0
```

Упрощая:
```
x^(4) - x = 0    (1)
```

### Шаг 2: Решение дифференциального уравнения

Характеристическое уравнение:
```
r⁴ - 1 = 0
```

Разложим: `r⁴ - 1 = (r² - 1)(r² + 1) = (r - 1)(r + 1)(r - i)(r + i)`

Корни:
```
r₁ = 1
r₂ = -1
r₃ = i
r₄ = -i
```

Общее решение:
```
x(t) = C₁e^t + C₂e^(-t) + C₃cos(t) + C₄sin(t)
```

### Шаг 3: Использование краевых условий

**Условие 1:** `x(0) = 0`
```
C₁ + C₂ + C₃ = 0    (2)
```

**Условие 2:** `x(T) = 0`
```
C₁e^T + C₂e^(-T) + C₃cos(T) + C₄sin(T) = 0    (3)
```

**Условие 3:** `x'(T) = 1`

Производная:
```
x'(t) = C₁e^t - C₂e^(-t) - C₃sin(t) + C₄cos(t)
```

При t = T:
```
C₁e^T - C₂e^(-T) - C₃sin(T) + C₄cos(T) = 1    (4)
```

**Условие 4:** Естественное граничное условие при t = 0

Поскольку x(0) фиксировано, но x'(0) может быть свободным, 
используем условие трансверсальности для производной высшего порядка.

Если x'(0) свободно, то:
```
[∂F/∂x''']|_(t=0) = 0
```

Но `F = x''² - x²` не зависит от x''', поэтому нужно учесть другой подход.

Альтернативно, если x'(0) не задано, используем условие:
```
[∂F/∂x'']|_(t=0) = 0, если x''(0) свободно
```

Однако обычно для таких задач требуется дополнительное условие.

**Условие 4:** `x''(0) = 0` (наиболее естественное для задач с фиксированными концами)

Вторая производная:
```
x''(t) = C₁e^t + C₂e^(-t) - C₃cos(t) - C₄sin(t)
```

При t = 0:
```
C₁ + C₂ - C₃ = 0    (5)
```

### Шаг 4: Решение системы уравнений

Из (2) и (5):
```
C₁ + C₂ + C₃ = 0
C₁ + C₂ - C₃ = 0
```

Вычитая: `2C₃ = 0`, следовательно `C₃ = 0`

Из (2): `C₁ + C₂ = 0`, следовательно `C₂ = -C₁`

Система упрощается до трех уравнений с тремя неизвестными:
- `C₂ = -C₁`
- `C₃ = 0`
- `C₁(e^T - e^(-T)) + C₄sin(T) = 0`
- `C₁(e^T + e^(-T)) + C₄cos(T) = 1`

Используя гиперболические функции:
```
2C₁sinh(T) + C₄sin(T) = 0    (6)
2C₁cosh(T) + C₄cos(T) = 1    (7)
```

Из (6): `C₄ = -2C₁sinh(T)/sin(T)`

Подставляя в (7):
```
2C₁cosh(T) - 2C₁sinh(T)cos(T)/sin(T) = 1
2C₁[cosh(T) - sinh(T)ctg(T)] = 1
```

Следовательно:
```
C₁ = 1/{2[cosh(T) - sinh(T)ctg(T)]}
C₂ = -C₁
C₄ = -2C₁sinh(T)/sin(T)
```

### Шаг 5: Упрощение выражения

Можно упростить:
```
cosh(T) - sinh(T)ctg(T) = cosh(T) - sinh(T)cos(T)/sin(T)
= [cosh(T)sin(T) - sinh(T)cos(T)]/sin(T)
= sin(T - iT)/sin(T) или использовать тригонометрические тождества
```

Альтернативно, используя формулы:
```
C₁ = sin(T)/{2[cosh(T)sin(T) - sinh(T)cos(T)]}
```

### Шаг 6: Финальное решение

```
x(t) = C₁(e^t - e^(-t)) + C₄sin(t)
x(t) = 2C₁sinh(t) + C₄sin(t)
```

где коэффициенты определены выше.

### Ответ:

Оптимальная функция `x*(t)` имеет вид:
```
x*(t) = 2C₁sinh(t) + C₄sin(t)
```

где:
- `C₁ = 1/{2[cosh(T) - sinh(T)ctg(T)]}`
- `C₄ = -2C₁sinh(T)/sin(T)`
- Коэффициенты зависят от T ∈ [0.1; 10.0]
