# Решение задачи Больца 2.1

**Условие:**
- Функционал: `J = ∫₀¹ (x'² + 2x)dt + x²(0) → extr`
- Нет фиксированных краевых условий

## Решение:

### Шаг 1: Уравнение Эйлера

Для задачи Больца с функционалом `J = ∫₀¹ F(x, x', t)dt + G(x(0), x(1))`,
где `F = x'² + 2x` и `G = x²(0)`.

Уравнение Эйлера (внутри интервала):
```
d/dt(∂F/∂x') - ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x' = 2x'`
- `∂F/∂x = 2`

Подставляем в уравнение Эйлера:
```
d/dt(2x') - 2 = 0
2x'' - 2 = 0
x'' = 1
```

### Шаг 2: Решение дифференциального уравнения

Общее решение:
```
x(t) = ∫∫ 1 dt dt = t²/2 + C₁t + C₂
```

### Шаг 3: Условия трансверсальности

Для задачи Больца условия трансверсальности имеют вид:
- В точке `t = 0`: `[∂F/∂x' - ∂G/∂x]|_(t=0) = 0`
- В точке `t = 1`: `[∂F/∂x' + ∂G/∂x]|_(t=1) = 0`

В данном случае `G = x²(0)`, поэтому:
- `∂G/∂x(0) = 2x(0)`
- `∂G/∂x(1) = 0` (G не зависит от x(1))

**Условие трансверсальности при t = 0:**
```
[∂F/∂x' - ∂G/∂x(0)]|_(t=0) = 0
[2x' - 2x(0)]|_(t=0) = 0
2x'(0) - 2x(0) = 0
x'(0) - x(0) = 0    (1)
```

**Условие трансверсальности при t = 1:**
```
[∂F/∂x' + ∂G/∂x(1)]|_(t=1) = 0
[2x' + 0]|_(t=1) = 0
2x'(1) = 0
x'(1) = 0    (2)
```

### Шаг 4: Использование условий трансверсальности

Из общего решения: `x(t) = t²/2 + C₁t + C₂`
```
x'(t) = t + C₁
```

**Условие (2):** `x'(1) = 0`
```
1 + C₁ = 0
C₁ = -1
```

**Условие (1):** `x'(0) - x(0) = 0`
```
C₁ - C₂ = 0
-1 - C₂ = 0
C₂ = -1
```

### Шаг 5: Экстремаль

```
x*(t) = t²/2 - t - 1
```

### Шаг 6: Определение типа экстремума

Вычислим значение функционала:
```
x*'(t) = t - 1
```

```
J[x*] = ∫₀¹ [(x*')² + 2x*]dt + x*²(0)
= ∫₀¹ [(t-1)² + 2(t²/2 - t - 1)]dt + (-1)²
= ∫₀¹ [(t² - 2t + 1) + (t² - 2t - 2)]dt + 1
= ∫₀¹ [2t² - 4t - 1]dt + 1
= [2t³/3 - 2t² - t]₀¹ + 1
= (2/3 - 2 - 1) + 1
= 2/3 - 2
= -4/3
```

Вторая вариация:
```
δ²J = ∫₀¹ (η')²dt + 2η²(0) > 0
```

для любой ненулевой функции `η(t)` с соответствующими условиями.

Следовательно, экстремаль доставляет абсолютный минимум.

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = t²/2 - t - 1
```

**Значение функционала:**
```
S_min = -4/3
S_max = +∞
```
