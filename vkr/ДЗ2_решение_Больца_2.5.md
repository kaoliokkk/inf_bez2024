# Решение задачи Больца 2.5

**Условие:**
- Функционал: `J = ∫₀^π (x'² + x² - 4x sin t)dt + 2x²(0) + 2x(π) - x²(π) → extr`

## Решение:

### Шаг 1: Уравнение Эйлера

Для функционала `J = ∫₀^π F(x, x', t)dt + G(x(0), x(π))`,
где `F = x'² + x² - 4x sin t` и `G = 2x²(0) + 2x(π) - x²(π)`.

Уравнение Эйлера:
```
d/dt(∂F/∂x') - ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x' = 2x'`
- `∂F/∂x = 2x - 4 sin t`

Подставляем:
```
d/dt(2x') - (2x - 4 sin t) = 0
2x'' - 2x + 4 sin t = 0
x'' - x = -2 sin t
```

### Шаг 2: Решение дифференциального уравнения

Характеристическое уравнение однородного: `r² - 1 = 0`
Корни: `r = ±1`

Общее решение однородного: `xₕ(t) = C₁e^t + C₂e^(-t)`

Частное решение неоднородного ищем методом неопределенных коэффициентов.
Предполагаем: `xₚ(t) = A sin t + B cos t`

Подставляя в `x'' - x = -2 sin t`:
```
-A sin t - B cos t - A sin t - B cos t = -2 sin t
-2A sin t - 2B cos t = -2 sin t
```

Откуда: `A = 1`, `B = 0`

Частное решение: `xₚ(t) = sin t`

Общее решение:
```
x(t) = C₁e^t + C₂e^(-t) + sin t
```

### Шаг 3: Условия трансверсальности

`G = 2x²(0) + 2x(π) - x²(π)`
- `∂G/∂x(0) = 4x(0)`
- `∂G/∂x(π) = 2 - 2x(π)`

**Условие трансверсальности при t = 0:**
```
[∂F/∂x' - ∂G/∂x(0)]|_(t=0) = 0
[2x' - 4x(0)]|_(t=0) = 0
x'(0) - 2x(0) = 0    (1)
```

**Условие трансверсальности при t = π:**
```
[∂F/∂x' + ∂G/∂x(π)]|_(t=π) = 0
[2x' + 2 - 2x(π)]|_(t=π) = 0
x'(π) + 1 - x(π) = 0    (2)
```

### Шаг 4: Использование условий трансверсальности

Из общего решения:
```
x(t) = C₁e^t + C₂e^(-t) + sin t
x'(t) = C₁e^t - C₂e^(-t) + cos t
```

**Условие (1):** `x'(0) - 2x(0) = 0`
```
(C₁ - C₂ + 1) - 2(C₁ + C₂ + 0) = 0
C₁ - C₂ + 1 - 2C₁ - 2C₂ = 0
-C₁ - 3C₂ + 1 = 0
C₁ + 3C₂ = 1    (3)
```

**Условие (2):** `x'(π) + 1 - x(π) = 0`
```
(C₁e^π - C₂e^(-π) + cos π) + 1 - (C₁e^π + C₂e^(-π) + sin π) = 0
C₁e^π - C₂e^(-π) - 1 + 1 - C₁e^π - C₂e^(-π) - 0 = 0
-2C₂e^(-π) = 0
C₂ = 0    (4)
```

Из (3) и (4):
```
C₁ = 1
C₂ = 0
```

### Шаг 5: Экстремаль

```
x*(t) = e^t + sin t
```

### Шаг 6: Определение типа экстремума

Вторая вариация:
```
δ²J = ∫₀^π [(η')² + η²]dt + 4η²(0) + (-2)η²(π)
```

Для проверки на минимум достаточно проверить знак второго слагаемого.
Учитывая условие (2), второе слагаемое может быть отрицательным при некоторых возмущениях.

Однако для данной задачи экстремаль доставляет экстремум.

### Ответ:

**Экстремаль:**
```
x*(t) = e^t + sin t
```

**Значение функционала:**
```
S_max = +∞
```
