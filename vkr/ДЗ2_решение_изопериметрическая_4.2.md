# Решение изопериметрической задачи 4.2

**Условие:**
- Функционал: `J = ∫₀¹ x'²dt → extr`
- Изопериметрическое ограничение: `∫₀¹ x dt = 0`
- Краевые условия: `x(0) = 1`, `x(1) = 0`

## Решение:

### Шаг 1: Метод множителей Лагранжа

Для изопериметрической задачи используем функционал:
```
J_λ = ∫₀¹ (F + λG)dt
```

где:
- `F = x'²` (подынтегральное выражение функционала)
- `G = x` (подынтегральное выражение ограничения)
- `λ` - множитель Лагранжа

Функционал Лагранжа:
```
J_λ = ∫₀¹ (x'² + λx)dt
```

### Шаг 2: Уравнение Эйлера

Для функционала `J_λ = ∫ (x'² + λx)dt` уравнение Эйлера:
```
d/dt(∂(x'² + λx)/∂x') - ∂(x'² + λx)/∂x = 0
```

Вычисляем частные производные:
- `∂(x'² + λx)/∂x' = 2x'`
- `∂(x'² + λx)/∂x = λ`

Подставляем:
```
d/dt(2x') - λ = 0
2x'' - λ = 0
x'' = λ/2
```

### Шаг 3: Решение дифференциального уравнения

Общее решение:
```
x(t) = ∫∫ (λ/2) dt dt = (λ/2)·t²/2 + C₁t + C₂ = λt²/4 + C₁t + C₂
```

### Шаг 4: Использование краевых условий и ограничения

**Условие 1:** `x(0) = 1`
```
C₂ = 1
```

**Условие 2:** `x(1) = 0`
```
λ/4 + C₁ + 1 = 0
C₁ = -λ/4 - 1    (1)
```

**Изопериметрическое ограничение:** `∫₀¹ x dt = 0`
```
∫₀¹ (λt²/4 + C₁t + 1)dt = 0
[λt³/12 + C₁t²/2 + t]₀¹ = 0
λ/12 + C₁/2 + 1 = 0    (2)
```

Подставляя (1) в (2):
```
λ/12 + (-λ/4 - 1)/2 + 1 = 0
λ/12 - λ/8 - 1/2 + 1 = 0
λ/12 - λ/8 + 1/2 = 0
(2λ - 3λ)/24 + 1/2 = 0
-λ/24 + 1/2 = 0
λ = 12
```

Из (1):
```
C₁ = -12/4 - 1 = -3 - 1 = -4
```

### Шаг 5: Экстремаль

```
x*(t) = 12t²/4 - 4t + 1 = 3t² - 4t + 1
```

### Шаг 6: Определение типа экстремума

Вычислим значение функционала:
```
x*'(t) = 6t - 4
```

```
J[x*] = ∫₀¹ (x*')²dt = ∫₀¹ (6t - 4)²dt
= ∫₀¹ (36t² - 48t + 16)dt
= [12t³ - 24t² + 16t]₀¹
= 12 - 24 + 16 = 4
```

Вторая вариация:
```
δ²J = ∫₀¹ (η')²dt > 0
```

для любой функции `η(t)` с `η(0) = η(1) = 0` и `∫₀¹ η dt = 0`.

Следовательно, экстремаль доставляет абсолютный минимум.

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = 3t² - 4t + 1
```

**Значение функционала:**
```
S_min = 4
S_max = +∞
```
