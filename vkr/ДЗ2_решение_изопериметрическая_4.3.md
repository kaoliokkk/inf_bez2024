# Решение изопериметрической задачи 4.3

**Условие:**
- Функционал: `J = ∫₀¹ x'²dt → extr`
- Изопериметрическое ограничение: `∫₀¹ x dt = 3`
- Краевые условия: `x(0) = 1`, `x(1) = 6`

## Решение:

### Шаг 1: Метод множителей Лагранжа

Функционал Лагранжа:
```
J_λ = ∫₀¹ (x'² + λx)dt
```

где `λ` - множитель Лагранжа.

### Шаг 2: Уравнение Эйлера

Уравнение Эйлера:
```
d/dt(2x') - λ = 0
2x'' - λ = 0
x'' = λ/2
```

### Шаг 3: Решение дифференциального уравнения

Общее решение:
```
x(t) = λt²/4 + C₁t + C₂
```

### Шаг 4: Использование краевых условий и ограничения

**Условие 1:** `x(0) = 1`
```
C₂ = 1
```

**Условие 2:** `x(1) = 6`
```
λ/4 + C₁ + 1 = 6
C₁ = 5 - λ/4    (1)
```

**Изопериметрическое ограничение:** `∫₀¹ x dt = 3`
```
∫₀¹ (λt²/4 + C₁t + 1)dt = 3
[λt³/12 + C₁t²/2 + t]₀¹ = 3
λ/12 + C₁/2 + 1 = 3
λ/12 + C₁/2 = 2    (2)
```

Подставляя (1) в (2):
```
λ/12 + (5 - λ/4)/2 = 2
λ/12 + 5/2 - λ/8 = 2
λ/12 - λ/8 + 5/2 = 2
(2λ - 3λ)/24 = 2 - 5/2
-λ/24 = -1/2
λ = 12
```

Из (1):
```
C₁ = 5 - 12/4 = 5 - 3 = 2
```

### Шаг 5: Экстремаль

```
x*(t) = 12t²/4 + 2t + 1 = 3t² + 2t + 1
```

### Шаг 6: Определение типа экстремума

Вычислим значение функционала:
```
x*'(t) = 6t + 2
```

```
J[x*] = ∫₀¹ (x*')²dt = ∫₀¹ (6t + 2)²dt
= ∫₀¹ (36t² + 24t + 4)dt
= [12t³ + 12t² + 4t]₀¹
= 12 + 12 + 4 = 28
```

Вторая вариация положительна, следовательно, экстремаль доставляет абсолютный минимум.

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = 3t² + 2t + 1
```

**Значение функционала:**
```
S_max = +∞
```
