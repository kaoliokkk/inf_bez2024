# Решение задачи простейшего вариационного исчисления 1.3

**Условие:**
- Функционал: `J = ∫₀¹ (x'² - x)dt → extr`
- Краевые условия: `x(0) = x(1) = 0`

## Решение:

### Шаг 1: Уравнение Эйлера

Для функционала `J = ∫ F(x, x', t)dt`, где `F = x'² - x`,
уравнение Эйлера:

```
d/dt(∂F/∂x') - ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x' = 2x'`
- `∂F/∂x = -1`

Подставляем в уравнение Эйлера:
```
d/dt(2x') - (-1) = 0
2x'' + 1 = 0
x'' = -1/2
```

### Шаг 2: Решение дифференциального уравнения

Общее решение:
```
x(t) = -t²/4 + C₁t + C₂
```

### Шаг 3: Использование краевых условий

**Условие 1:** `x(0) = 0`
```
C₂ = 0
```

**Условие 2:** `x(1) = 0`
```
-1/4 + C₁ + 0 = 0
C₁ = 1/4
```

### Шаг 4: Экстремаль

```
x*(t) = -t²/4 + t/4 = (-t² + t)/4 = t(1-t)/4
```

### Шаг 5: Определение типа экстремума

Вычислим значение функционала на экстремали:
```
x*'(t) = -t/2 + 1/4 = (1 - 2t)/4
```

```
J[x*] = ∫₀¹ [(x*')² - x*]dt = ∫₀¹ [((1-2t)/4)² - t(1-t)/4]dt
= ∫₀¹ [(1-2t)²/16 - t(1-t)/4]dt
= ∫₀¹ [(1 - 4t + 4t²)/16 - (t - t²)/4]dt
= ∫₀¹ [1/16 - t/4 + t²/4 - t/4 + t²/4]dt
= ∫₀¹ [1/16 - t/2 + t²/2]dt
= [t/16 - t²/4 + t³/6]₀¹
= 1/16 - 1/4 + 1/6
= (3 - 12 + 8)/48 = -1/48
```

Проверим вторую вариацию:
```
δ²J = ∫₀¹ (η')²dt > 0
```

для любой ненулевой функции `η(t)` с `η(0) = η(1) = 0`.

Следовательно, экстремаль доставляет абсолютный минимум.

### Шаг 6: Значение функционала

```
S_min = J[x*] = -1/48
```

Максимум не достигается.

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = -t²/4 + t/4 = t(1-t)/4
```

**Значение функционала:**
```
S_min = -1/48
S_max = +∞
```
