### Дифференциальная геометрия

**Вопрос:** Натуральным параметром кривой называется ...

**Ответ:** Параметр, который принимает только положительные значения. Норма производной от радиус-вектора $\mathbf{r}(t)$, которая при всех значениях параметра постоянна и равна единице.

**Вопрос:** Бинормаль и нормаль лежат в:

**Ответ:** нормальной плоскости

**Вопрос:** Чему равна средняя кривизна сферы радиуса $R$?

**Ответ:** $\frac{1}{R}$

**Вопрос:** Какие из перечисленных формул входят в состав формул Френе–Серре?

**Ответ:** $\frac{d\mathbf{n}}{dl} = -k\mathbf{b}$, $\frac{d\mathbf{b}}{dl} = -\kappa\mathbf{n}$

**Вопрос:** Если заданы первая и вторая квадратичные формы с матрицами $G$ и $H$ соответственно, то средняя кривизна вычисляется как:

**Ответ:** $\frac{1}{2}Tr(H \cdot G^{-1})$

**Вопрос:** Плоскость, проходящая через данную точку гладкой кривой ортогонально касательной, называется:

**Ответ:** нормальной плоскостью

### Функциональный анализ

**Вопрос:** Какому линейному оператору можно поставить в соответствие матрицу, которая полностью определяет действия этого оператора на элемент линейного пространства?

**Ответ:** Линейному оператору, переводящему элемент линейного пространства в элемент того же пространства.

**Вопрос:** Какие свойства выполняются для элементов $a, b, c \in L$ и $\alpha, \beta \in \mathbb{R}$ линейного пространства $L$?

**Ответ:**
- $(a + b) + c = a + (b + c)$ – ассоциативность по сложению;
- $a + b = b + a$ – коммутативность по сложению;
- $\exists 0_L \in L$ такой, что $a + 0_L = 0_L + a = a$ – нейтральный элемент по сложению;
- $\forall a \in L$ $\exists -a \in L$ такой, что $a + (-a) = (-a) + a = 0_L$ – обратный элемент по сложению;
- $\alpha(a + b) = \alpha a + \alpha b$ – дистрибутивность по сложению;
- $(\alpha + \beta)a = \alpha a + \beta a$ – дистрибутивность по умножению.
