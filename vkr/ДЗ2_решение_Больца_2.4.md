# Решение задачи Больца 2.4

**Условие:**
- Функционал: `J = ∫₀¹ (x'² + x²)dt - 2x(1)sh(1) → extr`
- Где `sh(1)` - гиперболический синус от 1

## Решение:

### Шаг 1: Уравнение Эйлера

Для функционала `J = ∫₀¹ F(x, x', t)dt + G(x(0), x(1))`,
где `F = x'² + x²` и `G = -2x(1)sh(1)`.

Уравнение Эйлера:
```
d/dt(∂F/∂x') - ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x' = 2x'`
- `∂F/∂x = 2x`

Подставляем:
```
d/dt(2x') - 2x = 0
2x'' - 2x = 0
x'' - x = 0
```

### Шаг 2: Решение дифференциального уравнения

Характеристическое уравнение: `r² - 1 = 0`
Корни: `r = ±1`

Общее решение:
```
x(t) = C₁e^t + C₂e^(-t)
```

Или в гиперболических функциях:
```
x(t) = A ch(t) + B sh(t)
```

где `ch(t)` и `sh(t)` - гиперболические косинус и синус.

### Шаг 3: Условия трансверсальности

`G = -2x(1)sh(1)`
- `∂G/∂x(0) = 0`
- `∂G/∂x(1) = -2sh(1)`

**Условие трансверсальности при t = 0:**
```
[∂F/∂x' - ∂G/∂x(0)]|_(t=0) = 0
[2x' - 0]|_(t=0) = 0
x'(0) = 0    (1)
```

**Условие трансверсальности при t = 1:**
```
[∂F/∂x' + ∂G/∂x(1)]|_(t=1) = 0
[2x' - 2sh(1)]|_(t=1) = 0
x'(1) - sh(1) = 0
x'(1) = sh(1)    (2)
```

### Шаг 4: Использование условий трансверсальности

Используем представление через гиперболические функции:
```
x(t) = A ch(t) + B sh(t)
x'(t) = A sh(t) + B ch(t)
```

**Условие (1):** `x'(0) = 0`
```
A·0 + B·1 = 0
B = 0
```

**Условие (2):** `x'(1) = sh(1)`
```
A sh(1) + 0·ch(1) = sh(1)
A sh(1) = sh(1)
A = 1
```

### Шаг 5: Экстремаль

```
x*(t) = ch(t)
```

### Шаг 6: Определение типа экстремума

Вычислим значение функционала:
```
x*'(t) = sh(t)
```

```
J[x*] = ∫₀¹ [(x*')² + x*²]dt - 2x*(1)sh(1)
= ∫₀¹ [sh²(t) + ch²(t)]dt - 2ch(1)sh(1)
```

Используя тождество `ch²(t) - sh²(t) = 1`, получаем `sh²(t) + ch²(t) = 2ch²(t) - 1`:
```
= ∫₀¹ [2ch²(t) - 1]dt - 2ch(1)sh(1)
```

Или используя `ch²(t) = (ch(2t) + 1)/2`:
```
sh²(t) + ch²(t) = ch(2t)
```

(Проверка: `ch(2t) = ch²(t) + sh²(t)` - верно)

```
J[x*] = ∫₀¹ ch(2t)dt - 2ch(1)sh(1)
= [sh(2t)/2]₀¹ - 2ch(1)sh(1)
= sh(2)/2 - 2ch(1)sh(1)
```

Используя `sh(2) = 2sh(1)ch(1)`:
```
J[x*] = 2sh(1)ch(1)/2 - 2ch(1)sh(1) = 0
```

Вторая вариация:
```
δ²J = ∫₀¹ [(η')² + η²]dt - 2sh(1)η²(1)
```

Учитывая условие трансверсальности, экстремаль доставляет экстремум.

### Ответ:

**Экстремаль:**
```
x*(t) = ch(t)
```

**Значение функционала:**
```
S_max = +∞
```
