# Решение задачи простейшего вариационного исчисления 1.6

**Условие:**
- Функционал: `J = ∫₀¹ (x'² + tx)dt → extr`
- Краевые условия: `x(0) = x(1) = 0`

## Решение:

### Шаг 1: Уравнение Эйлера

Для функционала `J = ∫ F(x, x', t)dt`, где `F = x'² + tx`,
уравнение Эйлера:

```
d/dt(∂F/∂x') - ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x' = 2x'`
- `∂F/∂x = t`

Подставляем в уравнение Эйлера:
```
d/dt(2x') - t = 0
2x'' - t = 0
x'' = t/2
```

### Шаг 2: Решение дифференциального уравнения

Интегрируем дважды:
```
x'(t) = ∫ (t/2)dt = t²/4 + C₁
x(t) = ∫ (t²/4 + C₁)dt = t³/12 + C₁t + C₂
```

Общее решение:
```
x(t) = t³/12 + C₁t + C₂
```

### Шаг 3: Использование краевых условий

**Условие 1:** `x(0) = 0`
```
C₂ = 0
```

**Условие 2:** `x(1) = 0`
```
1/12 + C₁ + 0 = 0
C₁ = -1/12
```

### Шаг 4: Экстремаль

```
x*(t) = t³/12 - t/12 = (t³ - t)/12 = t(t² - 1)/12
```

### Шаг 5: Определение типа экстремума

Вычислим значение функционала на экстремали:
```
x*'(t) = t²/4 - 1/12 = (3t² - 1)/12
```

```
J[x*] = ∫₀¹ [(x*')² + tx*]dt
= ∫₀¹ [((3t²-1)/12)² + t(t³-t)/12]dt
= ∫₀¹ [(9t⁴ - 6t² + 1)/144 + (t⁴ - t²)/12]dt
= ∫₀¹ [(9t⁴ - 6t² + 1)/144 + (12t⁴ - 12t²)/144]dt
= ∫₀¹ [(9t⁴ - 6t² + 1 + 12t⁴ - 12t²)/144]dt
= ∫₀¹ [(21t⁴ - 18t² + 1)/144]dt
= [21t⁵/5 - 18t³/3 + t]/144 |₀¹
= [21/5 - 6 + 1]/144
= [(21 - 30 + 5)/5]/144
= [-4/5]/144
= -4/720
= -1/180
```

Проверим вторую вариацию:
```
δ²J = ∫₀¹ (η')²dt > 0
```

для любой ненулевой функции `η(t)` с `η(0) = η(1) = 0`.

Следовательно, экстремаль доставляет абсолютный минимум.

### Шаг 6: Значение функционала

```
S_min = J[x*] = -1/180
```

Максимум не достигается.

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = (t³ - t)/12
```

**Значение функционала:**
```
S_min = -1/180
S_max = +∞
```
