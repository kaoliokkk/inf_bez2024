# Решение задачи простейшего вариационного исчисления 1.1

**Условие:**
- Функционал: `J = ∫₀¹ x'²dt → extr`
- Краевые условия: `x(0) = 1`, `x(1) = 0`

## Решение:

### Шаг 1: Уравнение Эйлера

Для функционала `J = ∫ F(x, x', t)dt`, где `F = x'²`,
уравнение Эйлера имеет вид:

```
d/dt(∂F/∂x') - ∂F/∂x = 0
```

Вычисляем частные производные:
- `∂F/∂x' = 2x'`
- `∂F/∂x = 0`

Подставляем в уравнение Эйлера:
```
d/dt(2x') - 0 = 0
2x'' = 0
x'' = 0
```

### Шаг 2: Решение дифференциального уравнения

Общее решение:
```
x(t) = C₁t + C₂
```

### Шаг 3: Использование краевых условий

**Условие 1:** `x(0) = 1`
```
C₂ = 1
```

**Условие 2:** `x(1) = 0`
```
C₁ + C₂ = 0
C₁ + 1 = 0
C₁ = -1
```

### Шаг 4: Экстремаль

```
x*(t) = -t + 1 = 1 - t
```

### Шаг 5: Определение типа экстремума

Вычислим значение функционала на экстремали:
```
J[x*] = ∫₀¹ (x*')²dt = ∫₀¹ (-1)²dt = ∫₀¹ 1 dt = 1
```

Проверим, является ли это минимумом. Рассмотрим возмущение:
```
x(t) = x*(t) + εη(t)
```
где `η(0) = η(1) = 0`.

Тогда:
```
J[x* + εη] = ∫₀¹ (x*' + εη')²dt = ∫₀¹ [(x*')² + 2εx*'η' + ε²(η')²]dt
= J[x*] + 2ε∫₀¹ x*'η'dt + ε²∫₀¹ (η')²dt
```

Второе слагаемое обращается в нуль в силу уравнения Эйлера (условие стационарности).

Третье слагаемое всегда положительно при `ε ≠ 0` и `η' ≠ 0`, следовательно:
```
J[x* + εη] ≥ J[x*]
```

Это означает, что экстремаль `x*(t) = 1 - t` доставляет абсолютный минимум.

### Шаг 6: Значение функционала

```
S_min = J[x*] = 1
```

Максимум не достигается (можно взять функции с произвольно большими производными).

### Ответ:

**Экстремаль (абсолютный минимум):**
```
x*(t) = 1 - t
```

**Значение функционала:**
```
S_min = 1
S_max = +∞
```
