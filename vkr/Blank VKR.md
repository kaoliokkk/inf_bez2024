## 7. Функциональный анализ

1. **Вопрос:** Какому линейному оператору можно поставить в соответствие матрицу, которая полностью определяет действия этого оператора на элемент линейного пространства?  
   **Ответ:** Линейному оператору, переводящему элемент линейного пространства в элемент того же пространства.

2. **Вопрос:** Какие свойства выполняются для элементов $a, b, c \in L$ и $\alpha, \beta \in \mathbb{R}$ линейного пространства $L$?  
   **Ответ:**
   - $(a + b) + c = a + (b + c)$ – ассоциативность по сложению;
   - $a + b = b + a$ – коммутативность по сложению;
   - $\exists 0_L \in L$ такой, что $a + 0_L = 0_L + a = a$ – нейтральный элемент по сложению;
   - $\forall a \in L$ $\exists -a \in L$ такой, что $a + (-a) = (-a) + a = 0_L$ – обратный элемент по сложению;
   - $\alpha(a + b) = \alpha a + \alpha b$ – дистрибутивность по сложению;
   - $(\alpha + \beta)a = \alpha a + \beta a$ – дистрибутивность по умножению.
